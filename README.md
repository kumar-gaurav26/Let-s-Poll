# 🗳️ Let's Poll

An interactive online polling and voting platform that allows users to create polls, participate in voting, and view live results in real time.

## 📌 Overview

Let's Poll is a mini-project designed to simplify the process of conducting online polls and surveys. The platform enables users to:

- Create custom polls
- Share polls with others
- Vote securely
- View live poll statistics and results
- Manage multiple polls efficiently

This project demonstrates concepts of full-stack web development including frontend UI design, backend integration, database management, and real-time data handling.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/kumar-gaurav26/Let-s-Poll.git
```

## 2️⃣ Navigate to Project Directory

```bash
cd Let-s-Poll
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Setup Environment Variables

Create a `.env` file in the root directory and add:

```env
PORT=5000
DB_URI=your_database_connection_string
JWT_SECRET=your_secret_key
```

---

# ▶️ Running the Project

## Start Backend Server

```bash
npm start
```

## Run Frontend

```bash
npm run dev
```

# 🔄 Workflow

1. User signs up or logs in
2. User creates a poll with multiple options
3. Poll link is shared with participants
4. Participants vote
5. Votes are stored in the database
6. Results are updated and displayed in real time
